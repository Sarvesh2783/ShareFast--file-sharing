# 🗄️ File Sharing Platform - Backend

This is the backend server of the File Sharing Platform. It handles file uploads, generates unique links, and serves files for download.

## ✨ Features

- Upload and store files
- Generate download URLs
- Serve files via API
- (Optional) File expiry or metadata

## 🛠 Tech Stack

- Node.js + Express.js
- Multer for file upload
- UUID for unique links
- dotenv for configuration
- MongoDB 
- nodemailer
- 

## 🚀 Getting Started

1.clone the repository
2.add .env file 
3.run the file
4.npm install 
5.npm run dev
